# Adivina el número

## Descripción

Este es un juego en el que el usuario debe adivinar un número aleatorio entre 1 y un número máximo según la dificultad seleccionada por el usuario.

## Requisitos

- Dart
- Flutter

## Instalación

1.Clonar el repositorio

```bash
git clone
```

2.Instalar las dependencias

```bash
flutter pub get
```

## Uso

```bash
flutter run
```
